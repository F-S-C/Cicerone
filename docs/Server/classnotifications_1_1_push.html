◆ create_payload()

static create_payload ( string  $type,
array  $parameters 
)
static

Get a payload array.

Parameters
string$typeThe type of notification message.
array$parametersThe parameters to be sent into the payload.
Returns
array An array containing the payload.
Precondition
$type should be a valid notification type (the ones defined as Push's constants).
Postcondition
The payload array, given \(n\in\mathbb{N}\) parameters, will have a field "message-type" containing the message type and \(n\) fields "param-i" containing the \(i\)-th parameter ( \(i\in\mathbb{N}, i=0,...,n-1\)). The following is an example of a valid payload array.
array(
"message-type" => "Message type",
"param-0" => "Parameter 0",
"param-1" => "Parameter 1",
...
"param-n" => "Parameter n"
);
